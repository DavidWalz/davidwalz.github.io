# Zensical

I just migrated from mkdocs-material to [Zensical](https://zensical.org/about/).
The switch was smooth and builds are super-fast now.

As background, Mkdocs 1.0 (on which mkdocs-material is based) has been unmaintained for a while, and the project has been in a state of limbo for the last year or so. 
Early this year, Mkdocs 2.0 was announced as a ground-up rewrite, that would break compatibility with the mkdocs-material and it's many plugins.
The maintainer of mkdocs-material has decided start a new static site generator which, while keeping compatibility.
I'm betting that the community will quickly converge on Zensical.
